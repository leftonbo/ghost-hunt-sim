# 👻 おばけ捕食シミュレーター

画面上の「おばけ👻」が「ニンゲン🧑」を追いかけて丸呑みにするシミュレーターです。  
食べられたニンゲンはおばけの体内で消化され、新たなおばけとして飛び出します。  
最終的に全員おばけになったらシミュレーション終了！

> かわいげのあるホラーがテーマ。怖いけどどこか楽しい雰囲気を大切にしています。

## 🎮 遊び方

1. `index.html` をブラウザで開く（Chrome / Edge / Firefox 推奨）
2. 操作パネルでおばけ数・ニンゲン数・速度を調整
3. 「▶ 開始」ボタンを押す
4. おばけがニンゲンを追いかける様子を眺める
5. 全ニンゲンがおばけになったらクリア！

## 🕹️ 操作

| 操作 | 説明 |
|------|------|
| ▶ 開始 | シミュレーションを開始 |
| ⏸ 一時停止 | 一時停止 / 再開のトグル |
| 🔄 リセット | 初期状態に戻す |
| 初期おばけ数 | 1〜10（デフォルト: 1） |
| 初期ニンゲン数 | 5〜100（デフォルト: 30） |
| 速度 | 0.5x〜3.0x（デフォルト: 1.0x） |

## 📋 情報表示

- 現在のおばけ数 / ニンゲン数
- 消化中のおばけ数
- 経過時間（mm:ss）
- 終了時メッセージ

## 🔧 技術仕様

- **シングル HTML ファイル**（HTML + CSS + JavaScript を1ファイルに収録）
- 外部ライブラリ不使用（Canvas API のみ）
- `requestAnimationFrame` ベースの 60fps ゲームループ
- update → draw の分離構造
- レスポンシブ対応（ウィンドウリサイズに追従）
